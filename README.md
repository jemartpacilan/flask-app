# Flask APP API

Python/Flask Backend Task by Irithm

# Requirements
```bash
- [Docker][docker]
- [Python][python] (>= 3.7)
```

## Installation

Use this command at the project to install the dependencies and start up the project

```bash
./up.sh
```

## Usage

Run app
```
./up.sh
```

Stop app
```
./down.sh
```

Print logs
```
./logs.sh
```
